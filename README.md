# Retail Store Sales Analysis

## Project Overview
This is an end-to-end retail sales analytics project using **Microsoft Excel, MySQL, and Tableau**. The project cleans and analyzes sales data to track revenue, seasonal trends, customer behavior (new vs returning, age groups), and discount effectiveness. Interactive **KPI dashboards** with filters for category, season, and gender provide actionable business insights.

## Tools & Skills
- **Microsoft Excel:** Data cleaning, calculated fields, pivot tables  
- **MySQL:** Queries, views, data aggregation  
- **Tableau:** KPI dashboards, filters, visualization  
- **Skills demonstrated:** Data analysis, customer behavior analytics, business insights

## Dataset
- Source: Kaggle Retail Store Sales Dataset  
- A **sample dataset** is included in the `data/` folder for reference.  
- (Full dataset not included due to size/license restrictions.)

## How to Use
1. Explore Excel files for data cleaning and pivot table analysis.  
2. Run MySQL scripts in a local database to generate queries and views.  
3. Open Tableau workbooks to view KPI dashboards and interact with filters.

## Outcome
The project demonstrates an **end-to-end analytics workflow** from raw data to actionable insights, supporting business decisions in retail sales performance and customer behavior analysis.
